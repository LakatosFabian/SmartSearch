# SmartSearch
Proiect "Smart search cu autocomplete" pentru Practica de specialitate

Aplicatie web fullstack pentru cautare de retete folosind:
- Node.js + Express
- MongoDB Atlas
- HTML, CSS, JavaScript

Functionalitati:
- cautare inteligenta cu sistem de ranking
- sugestii de autocomplete
- fuzzy matching (toleranta la greseli de tastare)
- cautare dupa ingrediente
